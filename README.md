# badge-up-cli

Create badges from the terminal, using [badge-up](https://www.npmjs.com/package/badge-up).

## Install

```
$ npm install --global badge-up-cli
```

## Usage

```
$ badge-up --help

  Usage
    $ badge-up <field1> <field2> <color>

  Example
    $ badge-up npm v1.0.0 "#007EC6"
```
